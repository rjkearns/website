Capstone Project
TransitBuilder, was a gamified pedigogical tool for teaching users about the resources required to operate a robust mass transit system in an urban environment. This project was implemented as a web application to maximize accessibility and consists of a Node.js server and a Leaflef-based client interface. This project digitizes and generalizes the transportation pedagogy of Jarret Walker, by putting users in the shoes of a transportation planner. I helped work on this project with Michael Lim, Nathan Pastor, and Ian Saad in our final semester at University of Puget Sound. Read our research paper here
